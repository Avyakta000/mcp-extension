Icon Requirements
================

Please add the following icon files to this directory:

1. icon16.png  - 16x16 pixels  - Toolbar icon
2. icon48.png  - 48x48 pixels  - Extension management page
3. icon128.png - 128x128 pixels - Chrome Web Store listing

Recommended Design:
- Purple/blue gradient background (#667eea to #764ba2)
- White wrench/tool icon in center
- Simple and recognizable design
- PNG format with transparency

You can create these using:
- Figma, Canva, or Photoshop
- Online icon generators
- AI image generators
